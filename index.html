MISSION_OBJECTIVE

โครงการนี้ถูกพัฒนาขึ้นเพื่อ สร้างองค์ความรู้และออกแบบตู้เลี้ยงจิ้งหรีดอัตโนมัติ ที่สามารถควบคุมสภาพแวดล้อม ทั้งอุณหภูมิและความชื้นได้อย่างแม่นยำผ่านเทคโนโลยี IoT

นอกจากการพัฒนาระบบแล้ว หัวใจสำคัญคือการ ถ่ายทอดเทคโนโลยี ให้กับกลุ่มเกษตรกร ชุมชนบ้านหนองใหญ่ เพื่อลดภาระงาน ลดต้นทุน และประเมินประสิทธิภาพการใช้งานจริงร่วมกับชุมชน (Participatory Action Research)

SYSTEM_ARCH

สถาปัตยกรรม ระบบ IoT

01

Microcontroller

บอร์ด NodeMCU ESP8266 ทำหน้าที่เป็นสมองกลสั่งการและประมวลผลข้อมูล พร้อมโมดูล Wi-Fi ในตัว

02

Smart Sensors

ติดตั้งเซนเซอร์ DHT11 ตรวจวัดค่าอุณหภูมิและความชื้นภายในตู้เลี้ยงอย่างแม่นยำ

03

Relay Control

ควบคุมการเปิด-ปิดอุปกรณ์ไฟฟ้าอัตโนมัติ เช่น พัดลมระบายอากาศและหลอดไฟอินฟราเรด

04

Cloud Dashboard

เชื่อมต่อข้อมูลขึ้น Cloud และแสดงผลบนแอปพลิเคชัน Blynk ให้ตรวจสอบและสั่งการได้ทันที
